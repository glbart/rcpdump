**rcpdump** - утилита для снифинга сетевых пакетов

Пока поддерживает только Mac OS. Захват пакетов происходит на канальном уровне через BPF (Berkeley Packet Filter) интерфейс.

```
Usage: rcpdump [OPTIONS] --interface-name <INTERFACE_NAME>

Options:
  -l, --list-interfaces
  -i, --interface-name <INTERFACE_NAME>
  -h, --help                             Print help
  -V, --version                          Print version
```
